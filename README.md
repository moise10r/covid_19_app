## Getting Started

To get a local copy up and running follow these simple example steps.

### Setup

```bash
git clone git@github.com:moise10r/covid_19_app.git
cd ./covid_19_app
```

### Install

```bash
npm install
```

### To Run it

```bash
npm start
```

### To Test it

```bash
npm run test
```


### Build

```bash
npm run build
```
